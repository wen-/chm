Weinre手机页面调试

http://people.apache.org/~pmuellr/weinre/docs/latest/Running.html

node.js安装weinre

npm install weinre -g

启动：

weinre --boundHost 192.168.20.198 --httpPort 8081 --reuseAddr true --readTimeout 1 --deathTimeout 5

boundHost:    -all-
httpPort:     8081
reuseAddr:    true
readTimeout:  1
deathTimeout: 5

 


使用：

在要调试的页面加入：

<script src="http://192.168.20.198:8081/target/target-script-min.js"></script>

访问：192.168.20.198:8081  在此界面上进行调试
