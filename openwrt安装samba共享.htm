Samba共享

安装samba36-server和luci-app-samba

[html] view plain copy
  1. opkg install samba36-server luci-app-samba  

简单介绍

Openwrt的samba配置是先读取/etc/config/samba，结合/etc/samba/smb.conf.template文件，生成配置文件/etc/samba/smb.conf，用户只需修改/etc/config/samba和/etc/samba/smb.conf.template文件,然后设置samba用户与密码。

编辑/etc/config/samba如下，其中config sambashare配置在luci页面编辑更方便。

[html] view plain copy
  1. config samba  
  2.     option workgroup 'WORKGROUP'  
  3.     option homes '1'  
  4.     option name 'snapgate-imx6'  
  5.     option description 'imx6-openwrt'  
  6.    
  7. config sambashare  
  8.     option name 'root' #网络显示的共享目录名字  
  9.     option path '/' #共享目录  
  10.     option users 'root'  
  11.     option read_only 'no'  
  12.     option guest_ok 'no'  
  13.     option create_mask '0700'  
  14.     option dir_mask '0700'  
  15.    


设置共享用户密码,刚开始口令文件是不存在的，先touch，再增加一个用户

[html] view plain copy
  1. touch /etc/samba/smbpasswd  
  2. smbpasswd  -a root  


修改samba模板,把字符集设置一下，启用root用户访问。

[html] view plain copy
  1. unixcharset = utf-8  
  2. #invalidusers = root
