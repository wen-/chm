  15. </div>
  16.  
  17. <div class="demo">
  18. 	<img src="http://dummyimage.com/50x50" alt="">
  19. </div>

实例CSS：

.demo{
	width: 100px;
	height: 100px;
	border: 2px solid #ddd;
	background: #f5f5f5;
	padding: 6px;
	float: left;
	margin-left: 20px;

	/*flex布局（作用于容器）*/
	display: flex;

	/*水平居中（作用于容器）*/
	justify-content: center;

	/*垂直居中（作用于容器）*/
	align-items: center;
}

.demo img{
	max-width: 100px;
	max-height: 100px;
	width: auto;
	height: auto;
}

五、左固定右自适应等高布局

演示截图：

HTML：

  1. <div class="demo">
  2. 	<div class="left">左边固定宽度为100px，这里设置了高度为auto</div>
  3. 	<div class="right">右边宽度自适应，并且左右两个区域是等高的，这里设置了高度为200px</div>
  4. </div>

CSS：

  1. .demo{
  2. 	/*flex布局（作用于容器）*/
  3. 	display: flex;
  4.  
  5. 	/*项目拉伸对齐，也就是所左边的高度为拉伸到和右边等高，默认是拉伸的*/
  6. 	/*align-items: stretch;*/
  7. }
  8.  
  9. .demo .left{
  10. 	/*左边固定宽度，必须设置其最小宽度和最大宽度*/
  11. 	width: 100px;
  12. 	min-width: 100px;
  13. 	max-width: 100px;
  14.  
  15. 	/*高度自由分配*/
  16. 	height: auto;
  17. 	background: #B4D3F7;
  18.  
  19. 	/*空白区域分配比例为0（作用于项目）*/
  20. 	flex-grow: 0;
  21. }
  22.  
  23. .demo .right{
  24. 	margin-left: 10px;
  25. 	width: auto;
  26. 	height: 200px;
  27. 	background: #F7E8B4;
  28.  
  29. 	/*空白区域分配比例为1（作用于项目）
  30. 	左右得到的空白比例为0:1，所以右边会分配到剩余的所有空白区域，
  31. 	左边成固定的宽度，右边为自适应宽度*/
  32. 	flex-grow: 1;
  33. }

六、左右固定中间自适应宽度底部对齐布局

上面的例子是左右布局的，相比较而言，双栏布局会做了，那么三栏布局也就不是问题了。先看实例图：

HTML：

  1. <div class="demo">
  2. 	<div class="left">左边固定宽度为100px，这里设置了高度为auto</div>
  3. 	<div class="center">中间宽度自适应，并且左中右两个区域是等高的，这里设置了高度为200px</div>
  4. 	<div class="right">右边固定宽度为150px，这里设置了高度为auto</div>
  5. </div>

CSS：

  1. .demo{
  2. 	/*flex布局（作用于容器）*/
  3. 	display: flex;
  4.  
  5. 	/*项目拉伸对齐，也就是所左边的高度为拉伸到和右边底部对齐*/
  6. 	align-items: flex-end;
  7. }
  8.  
  9. .demo .left{
  10. 	/*左边固定宽度，必须设置其最小宽度和最大宽度*/
  11. 	width: 100px;
  12. 	min-width: 100px;
  13. 	max-width: 100px;
  14.  
  15. 	/*高度自由分配*/
  16. 	height: auto;
  17. 	background: #B4D3F7;
  18.  
  19. 	/*空白区域分配比例为0（作用于项目）*/
  20. 	flex-grow: 0;
  21. }
  22.  
  23. .demo .center{
  24. 	margin: 0 10px;
  25. 	width: auto;
  26. 	height: 200px;
  27. 	background: #F7E8B4;
  28.  
  29. 	/*空白区域分配比例为1（作用于项目）
  30. 	左右得到的空白比例为0:1，所以右边会分配到剩余的所有空白区域，
  31. 	左边成固定的宽度，右边为自适应宽度*/
  32. 	flex-grow: 1;
  33. }
  34.  
  35. .demo .right{
  36. 	/*右边固定宽度，必须设置其最小宽度和最大宽度*/
  37. 	width: 150px;
  38. 	min-width: 150px;
  39. 	max-width: 150px;
  40.  
  41. 	/*高度自由分配*/
  42. 	height: auto;
  43. 	background: #CBFFD2;
  44.  
  45. 	/*空白区域分配比例为0（作用于项目）*/
  46. 	flex-grow: 0;
  47. }
