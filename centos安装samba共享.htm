centos安装samba共享

方法/步骤

  1. 最小化安装，很简单，下一步就可以

  2. 通过编辑文件修改网络配置

    cd /etc/sysconfig/network-scripts，

    运行ls命令会看到 ifcfg-enp4s0(每台电脑都不一样，我安装的另一台就是ifcfg-eno1,前半部分ifcfg-en是不变的)

    然后运行

    vi   ifcfg-enp4s0

  3. 需要修改的配置见下图

  4. 查看DNS地址

  5. 重启网络

    systemctl  restart network

  6. Centos7系统默认最小化安装的并没有nano、vim、wget、curl、ifconfig、lsof命令，这里首先安装一下：

    yum -y install nano vim wget curl net-tools lsof

  7. 7

    安装Samba

    # yum -y install samba samba-client samba-common

  8. 8

    关闭SELINUX，之后重启下

  1. 创建共享目录，人事和财务

  2. 配置内核参数

  3. 进入samba目录下，备份它的配置文件，并修改配置文件

  4. 创建用户并分配权限

  5. 查看设置的权限是否生效

  6. 重启samba服务并添加到开机启动

  7. 查看samba服务状态

  8. 将samba服务添加防火墙规则内，否则其他电脑无法访问，并重启防火墙
