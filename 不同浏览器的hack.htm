CSS hack写法：

.e{
color:#FFF; /* FF,OP */
[;color:#0F0;] /* Sa,CH */

#element {
    color: green\0/IE8+9; /* IE8+9  */
}
:root #element { color:pink \0/IE9; }  /* IE9 */
  1. background-color:red\0;IE8和IE9都支持；
  2. background-color:blue\9\0; 仅IE9支持；

color:#FFF\0; /*IE8、IE9*/
color:#FFF\9; /*IE6、7、8*/
*color:#FF0; /* IE7、6 */
_color:#F00; /* IE6 */
}
:root .e{color:#000;} /*IE9*/(:root CSS3，伪类指根，即HTML{})

注意hack的书写顺序。

 ! $ & * ( ) = % + @ , . / ` [ ] # ~ ? : <  > | */
