与attr()方法完全类似，jQuery提供了css()方法来直接获取、设置元素的样式风格。该方法的使用几乎和attr()一样，例如可以通过css(name)来获取某种样式风格的值，通过css(properties)列表来同时设置元素的多种样式，通过css(name,value)来设置元素的某种样式。
另外值得一提的是，css()方法提供了透明度的opccity属性，并且解决了浏览器的兼容性问题，不需要针对IE和Firefox分别开发。opacity属性的值取值范围为0.0~1.0，如：
$(function(){
       $("img").mouseover(fA)                      //代码形式1
function fA(){
              $(this).css("opacity","0.5");
};
$("img").mouseout(function(){            //代码形式2
       $(this).css("opacity","1.0");
});
});
另外，还可以通过hasClass(name)方法来判断某个元素是否设置了某个CSS类别，如果设置了则返回true，否则为false。例：
$("li:last").hasClass("myClass")
上述代码等同于：
$("li:last").is(".myClass")
