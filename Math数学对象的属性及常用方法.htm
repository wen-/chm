Math对象提供除加、减、乘、除以外的一些数学运算，如对数、平方根等，它是静态对象。

1. 主要属性
Math对象有6个属性，它们是数学中经常用到的几个常数，如下：
・ 常数e
・ 10的自然对数ln10
・ 2的自然对数ln2
・ PI：取值3.14159
・ 1/2的平方根：SQRT1/2
・ 2的平方根：SQRT2

2. 主要方法
主要的几个方法如下：
・ 绝对值：abs( )
・ 正弦余弦值：sin( )和cos( )
・ 反正弦反余弦：asin( )和acos( )
・ 正切反正切：tan( )和atan( )
・ 四舍五入：round( )
・ 平方根：sqrt( )
・ 基于几次方的值：Pow(base,exponent)
