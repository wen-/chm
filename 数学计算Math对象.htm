Math对象提供除加、减、乘、除以外的一些数学运算，如对数、平方根等，它是静态对象。

1. 主要属性
Math对象有6个属性，它们是数学中经常用到的几个常数，如下：
・ 常数e
・ 10的自然对数ln10
・ 2的自然对数ln2
・ PI：取值3.14159
・ 1/2的平方根：SQRT1/2
・ 2的平方根：SQRT2

2. 主要方法
主要的几个方法如下：
・ 绝对值：abs( )
・ 正弦余弦值：sin( )和cos( )
・ 反正弦反余弦：asin( )和acos( )
・ 正切反正切：tan( )和atan( )
・ 四舍五入：round( )
・ 平方根：sqrt( )
・ 基于几次方的值：Pow(base,exponent)

min()和max()方法用来反回一组数中的最大值或最小值，均可接受任意多个参数。
var iMax=Math.max(18,78,14,54);
var iMin=Math.min(18,78);

小数转为整数的方法分别是：ceil()、floor()、round()。ceil()向上舍入，floor()向下舍入，round()四舍五入。
var iNum=round(1.4);

Math对象另一个非常实用的方法是生成随机数的radom()方法，该方法返回一个0~1之间的随机数，不包括0和1。
返回一个1到100的随机数：
var iNum=Math.floor(Math.random()*100+1);
返回一个2~98的数字，第一个值为2：
var iNum=Math.floor(Math.random()*98+2);

返回x的绝对值：
Math.abs(x);

返回x的y次方：
Math.pow(x,y);
