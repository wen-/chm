vmware安装openwrt

1、下载openwrt镜像：http://downloads.openwrt.org/chaos_calmer/15.05/x86/generic/openwrt-15.05-x86-generic-combined-ext4.img.gz

解包得到里面的openwrt-15.05-x86-generic-combined-ext4.img

（冒似ext的包才可以成功启动）

 

2、将openwrt镜像转换为虚拟机磁盘文件格式vmdk

centos系统下使用qemu-utils命令转换(centos7为：qemu-img)

yum -y install qemu-utils   安装(centos7为：qemu-img)

qemu-img convert -f raw openwrt-15.05-x86-generic-combined-ext4.img -O vmdk openwrt-15.05-x86-generic-combined-ext4.vmdk    转换

将转换后的.vmdk文件复制到window磁盘上

 

3、新建虚拟机

选--自定义--硬件兼容9.0--稍后安装操作系统--操作系统luinx(其它luinx3.x内核)--命名保存--处理器1--内存256M--网络连接NAT--SSIC控制器（LSI Logic）--磁盘类型(IDE)--使用现有虚拟磁盘（选择之前复制过来的.vmdk）--自定义硬件（删除usb、光驱、声卡、打印机）--完成

（修改虚拟机中配置文件指向openwrt.vmx）

4、启动虚拟机

会停留在：br-lan: port 1(eth0) entered forwarding state      按下回车就可以了

 

5、配置网络

执行：vi /etc/config/network    编辑网络配置

修改'lan'配置（查看虚拟机--编辑--虚拟网络编辑器--看vmnet8的ip（192.168.27.0）段(查看DHCP设置128-254)）

将　ipaddr　的值修改为：vmnet8中的DHCP段中的一个如：192.168.27.128

esc :wq 保存退出，重启

在客户机浏览器中输入192.168.27.128就能看到路由本地界面了，但此时上不了外网。

 

6、配置上外网

网上说的修改wan口什么之类都是狗屁，wan口只要保留默认proto 'dhcp'，不用修改。

右键虚拟机--设置--添加网络适配器--桥接模式--完成

重启虚拟机，就可以上外网了，接着安装luci的中文包（命令行操作或进路由管理界面操作均可）

① opkg update  更新库

② opkg install luci-i18n-base-zh-cn  安装   有的包可能是luci-i18n-chinese，可能是版本不一样吧。

路由管理界面操作为--system--software--先点update lists更新（也可不更新）--download and install package: 在这里输入包名（luci-i18n-base-zh-cn ） 点OK安装

 

7、进入路由管理界面（默认无密码）--system--system--language and style--language(普通话)

至此大功告成！
