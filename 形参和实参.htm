形参和实参

实参多于形参时自动省略，少于则用undefined代替。

实参对象：arguments

最重要的用处是让函数可以操作任意数量的实参。

例获取传入参数的最大数：

function max(){

var max = Number.NEGATIVE_INFINITY;

for(var i=0;i<arguments.length;i++){

if(arguments[i] > max){max = arguments[i];}

}

return max;

}

var largest = max(1,2,3,4,5,6);

像这种可以接受任意个数的实参的函数被称为：“不定实参函数”
