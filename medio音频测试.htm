Installation(安装插件)

cordova plugin add cordova-plugin-media
 
调用：
document.addEventListener("deviceready", onDeviceReady, false); 
function onDeviceReady() { 
console.log(Media);
function playAudio(url) { 
 
    var my_media = new Media(url, 
        // success callback 
        function () { 
            console.log("playAudio():Audio Success"); 
        }, 
        // error callback 
        function (err) { 
            console.log("playAudio():Audio Error: " + err); 
        } ); 
        // Play audio 
        my_media.play(); 
    }
}
 
关于url路径的说明：
默认文件是从www/下开始的，如：www/audio/game_music.mp3文件，则url='audio/game_music.mp3'。
在android系统下，路径需要加上："/android_asset/www/" + url,
最终的路径也就是：/android_asset/www/'audio/game_music.mp3
