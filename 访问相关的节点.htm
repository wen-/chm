要访问<html/>元素（这是该文件的document元素），你可以使用document的documentElement特性：
var oHtml = document.documentElement;

　　如果你想取得<head/>和<body/>元素，使用下面的代码：
var oHead = oHtml.childNodes[0];
var oBody = oHtml.childNodes[1];
　　这其实是NodeList在JavaScript中的简便实现。实际上正式的从childNodes列表中获取子节点的方法是使用item()方法：
var oHead = oHtml.childNodes.item(0);
var oBody = oHtml.childNodes.item(1);

　　你还可以通过使用childNodes.length特性来获取子节点的数量：
alert(oHtml.childNodes.length);
　　HTML DOM定义了document.body作为指向<body/>元素的指针：
var oBody = document.body;
