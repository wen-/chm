事件流是指事件发生的顺序。分为冒泡型事件和捕获型事件。IE使用冒泡型事件，Netspace Navigator则使用捕获型事件。DOM同时支持冒泡型和捕获型，但是，捕获型事件先发生。

js阻止默认行为(window.event.returnValue = false; 与e.preventDefault();)

function stopDefault( e ) {  
if ( e && e.preventDefault ){

    e.preventDefault();      

}else{  
    window.event.returnValue = false; 

}          

return false;  

}
