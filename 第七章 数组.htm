第七章 数组

所有的数组都是对象

创建数组：

1、数组直接量

var empty = [];

var primes = [1,2,3,4];

var misc = [1,true,"2",3];

省略数组中的某个值会被undefined代替；

var undefs = [,,];//数组有两个元素都是undefined;允许有可选结尾的逗号。

2、调用构造函数创建数组

var a = new Array();//创建一个空数组；

var b = new Array(10);//创建指定长度的数组；

var c = new Array(1,2,3,4,"5");//创建指定数组元素的数组；
