phonegap Android调试

安装运行 weinre

首先，安装 weinre：

1. 安装 weinre
 > npm install weinre

去到对应的安装目录打开\node_modules\.bin文件夹行运行 weinre.cmd：

2. 运行 weinre 两个参数第一个是端口号，第二个是IP地址
 weinre --httpPort 8081 --boundHost 192.168.1.144
 
3. 打开谷歌浏览器
输入：192.168.1.144：8081
Access Points
debug client user interface:http://192.168.1.144:8081/client/#anonymous
 
Targets（这里显示链接到调试的设备，如果为none则没有设备链接）
  • 192.168.1.144 [channel: t-15 id: anonymous] - file:///android_asset/www/index.html
 
Clients （多人调试则后面ID不同）
 
  • 192.168.1.144 [channel: c-16 id: anonymous]

客户端（应用端）设置

4. 在要调试的页面引入js文件
 <script src="http://192.168.1.144:8081/target/target-script-min.js#anonymous"></script>
#号后面的id可自定义，保持和浏览器打开的一致
 
5. 更改配置文件config.xml
 <access origin="http://192.168.1.144:8081" />
允许访问外部域。
