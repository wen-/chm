将数值型转换为字符串类型，可以直接用加号“+”将数值加上一个长度为零的空字符串，或者用toString()方法。例：
<script type="text/javascript">
var a = 3;
var b = a+"";
var c =a.toString();
</script>
　　对于数值型转为字符串，如果使用toString()方法，还可以加入参数，进行进制的转换。如a.toString(2)转为二进制。

　　对于字符串转数值，两种方法分别是parseInt()和parseFloat()，分别转为整数和浮点数。只有字符类型才能调用这两种方法。利用parseInt()方法，同样可以实现进制的转换。例：document.write(parseInt("AF"16));显示为175。 parseFloat()方法与parseInt()方法类似。
