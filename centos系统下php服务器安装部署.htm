centos系统下php服务器安装部署

PHP服务器安装部署

 

一、    系统环境配置

1．    服务器操作系统centos6（6.0 - 6.6）

2．    apache版本2.2

3．    php版本5.3.3

二、    Apache安装

命令行输入

 [root@localhost ~]# yum -y install httpd

让 Apache 随系统启动：

[root@localhost ~]# chkconfig --levels 235 httpd on

配置完毕，启动 Apache：

[root@localhost ~]# /etc/init.d/httpd start

此时已经可以访问你的服务器，不出意外的话，能够看到 “Apache 2 Test Page powered by CentOS” 的测试页面。注意，如果其他机器访问这台服务无法显示这个页面，而直接在这台服务器上可以访问的话，一般情况下是 CentOS 自带的防火墙禁止了。你只需要进入防火墙，将 “WWW” 对应的 “80” 端口打开即可。

 

注意：在 CentOS 中 Apache 的默认根目录是 /var/www/html，配置文件 /etc/httpd/conf/httpd.conf。其他配置存储在 /etc/httpd/conf.d/ 目录。

 

打开配置文件/etc/httpd/conf/httpd.conf 找到AllowOverride None修改为AllowOverride All

 

三、    PHP安装

安装php及相关模块：

[root@localhost ~]# yum -y install php php-gd php-imap php-ldap php-odbc php-pear php-xml php-xmlrpc curl curl-devel

重启apache模块才能生效：

[root@localhost ~]# /etc/init.d/httpd restart

测试 PHP 相关信息

　　这步实际上可以省略，但是为了测试是否安装成功，你可以新建一个 index_test.php 页面进行测试，页面内容如下：

<?php
phpinfo();
?>

此时你可以访问你的站点地址，例如 “http://192.168.1.2/index_test.php”，查看是否能看到相关的 PHP 信息。

看到这样的图，就说明 PHP 安装成功了。

 

PHP服务器中依赖到了两个功能模块，一个是curl（向IM服务器请求数据）,另一个是mcrypt（对请求参数进行加密）。

1.   curl模块在通过上面的安装命令已经安装好了，只需要将SElinux运行模式改为permissive,因为在强制模式（默认）下，如果IM服务器地址带端口号的情况下是请求不到数据的。修改方法如下(重启生效)：

打开/etc/sysconfig/selinux 把SELINUX=enforcing改为SELINUX=permissive

2.   mcrypt模块在源上是没有的，所以无法通过yum来安装。只能通过源码包编译安装。

第一步：

libmcrypt-2.5.8.tar.gz

mhash-0.9.9.9.tar.gz

mcrypt-2.6.8.tar.gz

按顺序安装三个包，

①

tar -axvf libmcrypt-2.5.8.tar.gz

cd libmcrypt-2.5.8

./configure && make && make install

②

tar -axvf mhash-0.9.9.9.tar.gz

cd mhash-0.9.9.9

./configure && make && make install

③

tar -axvf mcrypt-2.6.8.tar.gz

cd mcrypt-2.6.8

LD_LIBRARY_PATH=/usr/local/lib ./configure && make && make install

 

第二步：

在这里已经把mcrypt.so模块生成好了，只需要执行两步操作：

①   复制mcrypt.so到 /usr/lib64/php/modules 文件夹下。

②   复制mcrypt.ini到 /etc/php.d 文件夹下。

 

四、    文件部署

将管理后台站点文件解压上传到 /var/www/html ，

修改文件权限：

cd /var/www/html

chmod 777 APP  //在这个文件夹下会生成runtime

cd /Public

chmod 777 Uploads   //这个文件夹是上传的图片及文件

 

重启apache。

 

站点文件简要说明

APP/Class

     PHPExcel.class.php    解释xls\xlsx

     PDES.class.php         加解密

     Wimage.class.php      验证码

APP/Conf

     address.php        IM服务器数据请求地址

     config.php             thinkphp常用配置

     verify.php             验证码配置

APP/data              

     Simhei.ttf             验证码字体
