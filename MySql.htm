MySql

window下手动安装mqsql：

1、官网下载社区版zip，5.7.7开始不包含data文件夹（初始化生成）

https://www.mysql.com/downloads/

2、初始化

mysqld --initialize-insecure --user=mysql  （无密码）

https://dev.mysql.com/doc/refman/5.7/en/data-directory-initialization-mysqld.html（初始化详细文档）

在Windows下，假如你的MySQL装在 D:\MySQL
就可以这样：
先切换到它的bin目录    cd D:\MySQL\bin\
D:\MySQL\bin>mysql -u root -p
接着输入你的root密码

接下来你就可以创建数据库、创建用户、创建修改表之类的操作（常用命令如下）。

查看现有数据库
mysql> show databases;

创建数据库(假如数据库名为 mydb)
mysql> create database mydb;

删除数据库（假如数据库名为 mydb）
mysql> drop database accounts;

使用数据库（假如使用数据库 mydb）
mysql> use mydb;
 
JSP连接数据库防止中文乱码：
在链接后面加上以下代码：
?useUnicode=true&characterEncoding=UTF-8
如：conn = DriverManager.getConnection("jdbc:mysql://localhost:3306/jd1109db2?useUnicode=true&characterEncoding=UTF-8","root","root");
 
关于密码：
默认root下是没密码的，要设置密码可执行：
mysqladmin -u root password 新密码
如已经有密码，要修改则执行：
mysqladmin -u root -p 旧密码 password 新密码
