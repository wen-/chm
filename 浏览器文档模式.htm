浏览器文档模式

最前沿标准（HTML5)

<!DOCTYPE HTML>

XHTML标准模式----必须是以<!DOCTYPE html开始,其它任何字符开始都会导致杂项（Quirks）模式,连<!---->、<?xml ...也不行

<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
<html xmlns="http://www.w3.org/1999/xhtml">
<head>
<meta http-equiv="Content-Type" content="text/html; charset=utf-8" />
<title>无标题文档</title>
</head>

近乎标准模式

<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Transitional//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd">
<html xmlns="http://www.w3.org/1999/xhtml">
<head>
<meta http-equiv="Content-Type" content="text/html; charset=utf-8" />
<title>无标题文档</title>
</head>

<html>
<head>
<meta http-equiv="X-UA-Compatible" content="IE=edge,chrome=1">  使用IE最高版本或谷歌内核
  <!-- Mimic Internet Explorer 7 -->
  <meta http-equiv="X-UA-Compatible" content="IE=EmulateIE7" />
EmulateIE7模拟IE7模式，代码主要有：IE=5,IE=7,IE=EmulateIE7,IE=8,IE=EmulateIE8,IE=9,IE=10,IE=edge，edge为最高版本模式  
<title>My webpage</title>
</head>
<body>
  <p>Content goes here.</p>
</body>
</html> 
 
 
配置服务器指定文档模式web.config
<?xml version="1.0" encoding="utf-8"?>
<configuration>
  <system.webServer>
    <httpProtocol>
      <customHeaders>
        <clear />
        <add name="X-UA-Compatible" value="IE=10" />
      </customHeaders>
    </httpProtocol>
  </system.webServer>
</configuration>
