返回值:jQuerykeydown(fn)

概述

在每一个匹配元素的keydown事件中绑定一个处理函数。

keydown事件会在键盘按下时触发。

参数

fnFunction

在每一个匹配元素的keydown事件中绑定的处理函数。

示例

描述:

在页面内对键盘按键做出回应，可以使用如下代码:

jQuery 代码:
$(window).keydown(function(event){
  switch(event.keyCode) {
    // ...
    // 不同的按键可以做不同的事情
    // 不同的浏览器的keycode不同
    // 更多详细信息:     http://unixpapa.com/js/key.html
    // ...
  }
});
