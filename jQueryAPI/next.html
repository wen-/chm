返回值:Array<Element(s)>prev + next

概述

匹配所有紧接在 prev 元素后的 next 元素

参数
prevSelector
任何有效选择器
nextSelector
一个有效选择器并且紧接着第一个选择器
示例
描述:
匹配所有跟在 label 后面的 input 元素
HTML 代码:
<form>
  <label>Name:</label>
  <input name="name" />
  <fieldset>
      <label>Newsletter:</label>
      <input name="newsletter" />
 </fieldset>
</form>
<input name="none" />
jQuery 代码:
$("label + input")
结果:
[ <input name="name" />, <input name="newsletter" /> ]
