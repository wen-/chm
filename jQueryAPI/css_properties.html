返回值:jQuerycss(properties)

概述

把一个“名/值对”对象设置为所有匹配元素的样式属性。

这是一种在所有匹配的元素上设置大量样式属性的最佳方式。

参数

propertiesMap

要设置为样式属性的名/值对

示例

描述:

将所有段落的字体颜色设为红色并且背景为蓝色。

jQuery 代码:
$("p").css({ color: "#ff0011", background: "blue" });

描述:

如果属性名包含 "-"的话，必须使用引号:

jQuery 代码:
$("p").css({ "margin-left": "10px", "background-color": "blue" });
