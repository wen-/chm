返回值:jQueryaddClass(class)

概述

为每个匹配的元素添加指定的类名。

参数
classString
一个或多个要添加到元素中的CSS类名，请用空格分开
示例
描述:
为匹配的元素加上 'selected' 类
HTML 代码:
<p>Hello</p>
jQuery 代码:
$("p").addClass("selected");
结果:
[ <p class="selected">Hello</p> ]
描述:
为匹配的元素加上 selected highlight 类
HTML 代码:
<p>Hello</p>
jQuery 代码:
$("p").addClass("selected highlight");
结果:
[ <p class="selected highlight">Hello</p> ]
