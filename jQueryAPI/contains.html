返回值:Array<Element(s)>:contains

概述

匹配包含给定文本的元素

参数
textString
一个用以查找的字符串
示例
描述:
查找所有包含 "John" 的 div 元素
HTML 代码:
<div>John Resig</div>
<div>George Martin</div>
<div>Malcom John Sinclair</div>
<div>J. Ohn
jQuery 代码:
$("div:contains('John')")
结果:
[ <div>John Resig</div>, <div>Malcom John Sinclair</div> ]
