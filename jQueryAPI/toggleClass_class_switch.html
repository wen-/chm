返回值:jQuerytoggleClass(class, switch)

概述

如果开关switch参数为true则加上对应的class，否则就删除。

参数
classString
要切换的CSS类名
switchBoolean
用于决定元素是否包含class的布尔值。
示例
描述:
每点击三下加上一次 'highlight' 类
HTML 代码:
<strong>jQuery 代码:</strong>
jQuery 代码:
  var count = 0;
  $("p").click(function(){
      $(this).toggleClass("highlight", count++ % 3 == 0);
  });
