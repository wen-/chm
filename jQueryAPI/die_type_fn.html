返回值:jQuerydie([type], [fn])

概述

jQuery 1.3新增。此方法与live正好完全相反。

如果不带参数，则所有绑定的live事件都会被移除。

你可以解除用live注册的自定义事件。

如果提供了type参数，那么会移除对应的live事件。

如果也指定了第二个参数function,则只移出指定的事件处理函数。

参数

type (可选)String

要解除绑定的live事件

fn (可选)Function

要解除绑定的特定处理函数

示例

描述:

给按钮解除click事件

jQuery 代码:
function aClick() {
      $("div").show().fadeOut("slow");
  }
  $("#unbind").click(function () {
      $("#theone").die("click", aClick)
  });
