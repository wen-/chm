返回值:jQueryremoveClass([class])

概述

从所有匹配的元素中删除全部或者指定的类。

参数
class (可选)String
一个或多个要删除的CSS类名，请用空格分开
示例
描述:
从匹配的元素中删除 'selected' 类
HTML 代码:
<p class="selected first">Hello</p>
jQuery 代码:
$("p").removeClass("selected");
结果:
[ <p class="first">Hello</p> ]
描述:
删除匹配元素的所有类
HTML 代码:
<p class="selected first">Hello</p>
jQuery 代码:
$("p").removeClass();
结果:
[ <p>Hello</p> ]
