box-shadow解读

语法：

box-shadow：<length> <length> <length> <length> || <color>

box-shadow: h-shadow /v-shadow /blur /spread /color /inset;(水平偏移、垂直偏移、模糊度、阴影大小、阴影颜色、内阴影)

相关属性：text-shadow

取值：

<length> <length> <length>? <length>? || <color>：
阴影水平偏移值（可取正负值）；阴影垂直偏移值（可取正负值）；阴影模糊值；阴影颜色

说明：

设置块阴影--可设置多重阴影

box-shadow: 0 0.3em 0.3em rgba(255, 254, 255, 0.6) inset, 0 -0.1em 0.3em rgba(0, 0, 0, 0.15) inset, 0 0.1em 3px #CC8400, 0 0.3em 1px #996300, 0 0.5em 5px rgba(0, 0, 0, 0.2);

 0 0.3em 0.3em rgba(255, 254, 255, 0.6) inset: 0为水平偏移；0.3em为垂直偏移；0.3em为模糊度；rgba()为颜色值和透明度；inset为内阴影。
