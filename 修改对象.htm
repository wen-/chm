1.为对象定义新方法
    可以用prototype属性为任何已有的类定义新方法。
    例如用toHexString()方法实现toString(16)进行16进制转换：
Number.prototype.toHexString = function () {
    return this.toString(16);
}

var iNum = 15;
alert(iNum.toHexString());//outputs “F”

2.重定义已有方法
    在JavaScript中，函数名只是指向函数的指针，因此可以轻易地使它指向其它函数。如修改本地方法toString()：
Function.prototype.toString = function () {
    return “Function code hidden”;
}

function sayHi() {
    alert(”hi”);
}

alert(sayHi.toString());//outputs “function code hidden”
