ciPHP2.2.x升级到3.0

1、更新system文件夹，替换根目录下的index.php文件（如有修改，将修改复制到新的index.php文件中）；

注意：如有任何自定义的文件在这些文件夹中，请复制；

2、所有类文件名首字母大写，适用于（application/libraries/、application/controllers/、application/models/ 所有php文件，扩展类：MY_Email ; 模型：Misc_model）；

3、替换config/mimes.php文件；

4、从config/autoload.php文件中移除所有 $autoload[‘core’]；

5、日志类被定义为核心类，请移动日志类：

application/libraries/Log.php -> application/core/Log.php
application/libraries/MY_Log.php -> application/core/MY_Log.php

6、session类的使用，之前版本的session都是存储在cookie中的，现在session类完全重写，不再存储在cookie中，需要服务端提供存储（数据库或服务端文件系统，默认为服务端文件系统来存储session会话）

文件系统：$config['sess_save_path']

mkdir /<path to your application directory>/sessions/
chmod 0700 /<path to your application directory>/sessions/
chown www-data /<path to your application directory>/sessions/

数据库（支持MySQL、PostgreSQL、Redis、Memcached）：以MySQL为例

配置

$config['sess_driver'] = 'database';
$config['sess_save_path'] = 'ci_sessions';

建表

CREATE TABLE IF NOT EXISTS `ci_sessions` (
        `id` varchar(40) NOT NULL,
        `ip_address` varchar(45) NOT NULL,
        `timestamp` int(10) unsigned DEFAULT 0 NOT NULL,
        `data` blob NOT NULL,
        PRIMARY KEY (id),
        KEY `ci_sessions_timestamp` (`timestamp`)
);

19、一些被移除的类或方法：

（1）以前不赞成使用SHA1库已被删除,修改你的代码使用PHP的原生SHA1()函数来生成一个SHA1哈希。
此外,加密库的sha1()方法被移除。

（2）EXT常数被移除。

（3）表情辅助函数被移除。

（4）Encrypt加密类被移除，取而代之的是Encryption加密类。

（5）购物车类被移除。

（6）使用mysqli驱动代替旧的mysql驱动。

（7）安全辅助函数被移除。

（8）文件辅助函数被弃用read_file()。

（9）字符串辅助函数被弃用repeater()、trim_slashes()、random_string() 类型中移除 ‘unique’ and ‘encrypt’。

（10）表单辅助函数被弃用form_prep()。

（11）邮件辅助函数被弃用。

（12）日期辅助函数被弃用standard_date()。

（13）HTML辅助函数被弃用nbs(), br()。

（14）分页类弃用anchor_class。

（15）session类userdata()取代all_userdata()。

（16）数据库方法add_column()

// Old usage:
$field = array(
        'new_field' => array('type' => 'TEXT')
);

$this->dbforge->add_column('table_name', $field, 'another_field');

// New usage:
$field = array(
        'new_field' => array('type' => 'TEXT', 'after' => 'another_field')
);

$this->dbforge->add_column('table_name', $field);

（17）input类修改is_cli_request()

// Old

// New
is_cli();
