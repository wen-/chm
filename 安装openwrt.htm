安装openwrt

1、安装编辑环境

yum install gcc-c++ binutils bzip2 flex python perl make find grep diff unzip gawk getopt subversion libz-dev libc ncurses-devel zlib-devel openssl

2、新建openwrt目录，并进入openwrt目录

mkdir openwrt ---- cd openwrt

3、安装git

yum install git

4、获取openwrt

git clone https://github.com/openwrt/openwrt.git

5、进入到git下来的openwrt目录

cd openwrt

6、更新脚本包

./scripts/feeds update -a

./scripts/feeds install -a

7、编译配置

make menuconfig    --此时如果缺包会提示
