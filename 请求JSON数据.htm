请求JSON数据

$.ajax({
  type: "POST",
  url: "json/json_1.json",
  dataType: "json",
  error: function(){
   console.log("msg");
  },
  success:function(json){
   alert(json.name)
  }
 });

注意事项：

请求的JSON数据必须格式正确，否则返回的数据不可操作：

正常格式：{"name":"aa"}

错误格式：{"name":aa}/{name:"aa"}/{name:aa}

比较一下json与jsonp格式的区别：

json格式：

{
 "message":"获取成功",
 "state":"1",
 "result":{"name":"工作组1","id":1,"description":"11"}
}

jsonp格式：

callback({
 "message":"获取成功",
 "state":"1",
 "result":{"name":"工作组1","id":1,"description":"11"}
})
