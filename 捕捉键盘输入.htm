对于Netscape浏览器，要想浏览器不断查看窗口中是否发生鼠标、键盘事件（包括MouseDown，MouseUp和KeyPress，KeyDown，KeyUp 事件），必须添加如下代码，否则将不会“捕捉”鼠标、键盘事件：
if (document.layers)
document.captureEvents(event.KEYPRESS);
　　对IE浏览器，则不需要，浏览器会自动不断检测是否发生了鼠标、键盘事件。

　　获取用户键盘输入的ASCII码值关键代码如下：
if (document.layers)
document.captureEvents(event.KEYPRESS);
if (document.layers)
var keyChar =e.which;
if (document.all)
var keyChar =event.keyCode;

　　字符在内存单元中是以ASCII码值的形式存储的。常用字符的ASCII码值如下：
・ 小写字母a～z的ASCII码值为97～122；
・ 大写字母A～Z的ASCII码值为65～90；
・ 数字0～9的ASCII码值为49～57；
・ 回车键的ASCII码值为13。

　　通过String.fromCharCode(ASCII码值)可以将ASCII码值转化为字符值。
