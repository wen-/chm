jQery概述

JQery是什么

目前jQuery主要提供如下功能
(1)访问页面框架的局部
(2)修改页面的表现
(3)更改页面的内容
(4)响应事件
(5)为页面添加动画
(6)与服务器异步交互
(7)简化常用的JavaScript操作

jQueryr的“$”

在jQuery中，“$”提供了丰富的功能，包括选择页面中一个或理一类元素、作为功能函数的前缀、window.onload的完善、创建页面的DOM节点等。

选择器

jQueryr中选择器的通用语法如下：
$(selector)或jQueryr(selector)
其中selector符合CSS3标准。在jQuery 中，美元符号“$”就等同于“jQuery”。
一些典型的jQueryr选择元素的例子：
$("#showDiv")
id选择器，相当于JavaScript中的document.getElementById("#showDiv")。
$(".SomeClass")
类别选择器，相当于CSS类别为“SoomeClass”的所有节点元素。
$("p:odd")
选择所有位于奇数行的<p>标记。几乎所有的标记都可以用“:odd”或“:even”来实现奇偶的选择。
$("td:nth-child(1)")
所有表格行的第一个单元格，就是第一列。
$("li>a")
子选择器，返回<li>标记的所有子元素<a>，不包括孙标记。
$("a[href$=pdf]")
选择所有超链接，并且这些超链接的href属性是以“pdf”结尾的。
使用选择器10.4章节将会详解。

功能函数前缀

例如用户在提交表单时，需要将文本框中的最前端和最末端的空格清理掉，JavaScript没有提供类似trim()的功能，引入jQuery后，便可以直接使用trim()函数：
sString=$.trim(sString);

解决window.onload函数的冲突

jQery中的ready()方法很好地解决了上述问题，它能够自动地将其中的函数在页面加载完成后运行，并且同一个页面中可以使用多个ready()方法，而且不相互冲突。
例如一个隔行变色的代码：
$(document).ready(function(){
$("table.datalist tr:nth-child(odd)").addclass("altrow");
});
对上述代码，jQuery还提供了简写，可以省略其中的“(document).ready”部分，代码如下：
$(function(){
$("table.datalist tr:nth-child(odd)").addclass("altrow");
});

创建DOM元素

jQuery使用“$”可以直接创建DOM元素，如：
var oNewP=$("<p>这是一个感人的故事</p> ")
以上代码等同于JavaScript中的如下代码：
var oNewP=document.createElement("p");
var oText=document.createTextNode("这是一个感人的故事");
oNewP.appendChild(oText);
另外，jQuery还提供了DOM元素的insertAfter()方法，例如：
$(function(){         //简写的ready()函数
var oNewP=$("<p>这是一个感人的故事</P");         //创建DOM元素
oNewP.insertAfter("#myTarget");         //用insertAfter()方法将新建的DOM元素插入到id为myTarget的标签后
});

自定义添加“$”

例如jQuery中并没有将一组表单元素设置为不可用的disable()方法，用户可以自定义该方法，代码如下：
$fn.disable=function(){         //为$添加一个方法。“$fn”是扩展jQuery所必需的
return this.each(funciton(){          //利用匿名函数定义这个方法，即用each()将调用此方法的每个元素设置属性
if(typeof this.disabled!="undefined") this.disabled=true;             //如果该属性未定义
});
}

解决“$”的冲突

如果开发者使用了别的类库框架，“$”符号可能会发生冲突，jQuery提供了noConflict()方法来解决“$”冲突问题，例如：
jQuery.noConfilct();
以上代码便可以让“$”按照其它JavaScript框架的方式运算。这时在jQuery中便不能再使用“$”，而必须使用“jQuery”。
