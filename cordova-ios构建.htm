cordova-ios构建

1、cordova create test io.cordova.test 应用名称；

2、cd test;

3、编辑好config.xml   及开发好相应的页面；

4、cordova platform add ios;

5、cordova install -g cordova-icon;

6、cordova install -g cordova-splash;

7、需要安装ImageMagick

8、http://brew.sh/index_zh-cn.html    安装brew包管理软件

/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

9、brew install ImageMagick

10、在项目根目录设计两个图png图片：icon.png\splash.png  大小约150*150、1280*1280

11、项目根目录执行：sudo cordova-icon

12、项目根目录执行：sudo cordova-splash

13、打开xcode,open项目：项目根目录\platforms\ios\应用名.xcodeproj

14、点击xcode菜单,添加开发者账户

15、选择打开的项目，右侧设置团队（添加的开发者账户）

16、连接IPHONE手机，在菜单window-device会看到对应的手机信息

(如device看不到手机，先拨出，重启虚拟机进入系统后再插入手机USB，手机不要锁屏，因锁屏问题重启XCODE，如出现iphone is busy: Processing symbol files,可能是没加载完，查看感叹号的进度条，需要几分钟加载）

17、菜单product项run

18、手机设置-通用-设备管理  允许开发者应用
